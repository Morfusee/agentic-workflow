01 What We're Deciding

? The Question

  • Replace or reduce Ghost Inspector for monitoring checks

  • Keep maintenance overhead low

Option A

03 TestDino

+ Pros

  • Historical visibility without building a dashboard
  • 5,000 executions/month on free tier
  • Faster to evaluate than self-hosted
  • Bridge option before owning infrastructure

− Cons

  • Alerts require paid plan (~$49/mo)
  • Free tier may limit to one team member
  • GitHub Actions dependency if used for scheduling
  • Vendor dependency for history and dashboard UX
Best fit: Use TestDino if the team wants the fastest path to historical visibility and is comfortable validating free-tier constraints during the POC.
Option B

04 Self-Hosted Playwright + Allure

+ Pros

  • Full control over runtime and scheduling
  • Datastore persists structured run metadata
  • Custom alerts to Slack, Chat, or email
  • Lower vendor lock-in, portable Docker stack

− Cons

  • Allure is not a full historical analytics platform
  • More moving parts to maintain
  • Initial setup more complex than TestDino
  • Artifact storage needs retention policies
Best fit: Use the self-hosted path if the team wants a deployable monitoring service it can own, customize, and run on a VPS or EC2 instance using Docker Compose.

POC

07 POC Scope & Success Criteria

⚙ What to Build

  • 2–3 Playwright checks (page load, element verify, CTA, form flow)
  • TestDino evaluation against free-tier limits
  • Dockerized runner + Compose + datastore + Allure
  • Failure alert to Slack, Chat, or email

✓ Success Criteria

  • ✓ Same checks run reliably in Playwright
  • ✓ Historical run info visible beyond latest report
  • ✓ Failed run produces clear alert with context
  • ✓ Monthly cost stays below Ghost Inspector
  • ✓ Team can decide based on evidence

Recommendation

POC First, Decide After

Run both paths with the same small monitoring suite. Compare ease of operation, cost, dashboard quality, alerting, and long-term ownership before committing.

⚡ Fastest Path

TestDino — test first if the team wants the fastest usable dashboard and can accept free-tier limits.

⚙ Long-Term Path

Self-hosted Playwright + Allure — better if the team wants an owned, deployable monitoring service with custom alerting.
