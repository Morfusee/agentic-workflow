# Session 1: IP Addressing & Subnetting Fundamentals

Established: user has no prior networking knowledge. Comfortable with Linux CLI. Goal is self-hosting via Proxmox on OVH KimSufi within ~1 month.

Covered: IPv4 address structure, public vs private IPs, CIDR notation (/32, /28, /24), subnet masks, network/gateway/broadcast addresses, OVH's convention of penultimate-IP-as-gateway, and how IP addressing maps to the Proxmox bridge + Additional IP pattern.

Next: Lesson 2 should cover network interfaces, bridges, and the `/etc/network/interfaces` file — the "wiring" layer that connects IPs to actual hardware.
